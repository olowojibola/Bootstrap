
Step 1: Initialize the project to create package json file. This turn
the project to npm project

>>> npm init -y

Step 2: Install using the version on the boostrap newest version

>>> npm install bootstrap@5.3.2

Step 3: For every actitivty, the update can be used to Github

>>> git add .  ## to add to Github
>>> git commit -m "Install Bootstrap" #commit news changes
>>> git push   ## Push it to Github
